📋 延迟卖出清单（盘前必看）

2026-05-07 股票投资

一、先做“身份确认”（只要是“是”，立刻进入延迟区）

如果这只票满足以下任意一条：

□ 我是提前埋伏买的

□ 买入理由是周期 / 补涨 / 再定价

□ 我当初说过一句话：“它迟早会涨”

□ 它属于ETF 或 ETF 成份股

👉 直接判定：延迟卖出，不准情绪操作

二、时间锁（最重要）

□ 买入未满 20 个交易日

👉 无条件禁止清仓
（哪怕浮亏、哪怕别的标的在暴涨）

你这次卖飞豫光金铅，100%就是死在这里

三、趋势锚检查（只看锚，不看个股）

给每只股提前写一个锚（你可以照抄）：

有色 / 冶炼 → 有色ETF / 白银ETF

能源 → 原油 / 能源ETF

科技 → 行业ETF / 纳指

地产 → 地产ETF / 港股地产

盘前只问一句：

□ 锚破位了吗？

否 → 禁止卖

是 → 进入“允许卖出区”（但仍不等于立刻卖）

四、“慢行情隔离器”（专治心烦）

如果你想卖出的原因是：

□ 横盘太久

□ 成交量太低

□ 看着别的票在飞

□ 账户里有被套票，想调仓

👉 这些理由全部无效
＝ 自动禁止卖出

五、起飞前 72 小时保护罩（关键）

如果出现任意一条：

□ 成交量温和放大

□ 连续 2–3 天小阳、不加速

□ 同板块开始补涨

👉 触发 72 小时保护

3 天内 禁止卖出

第 4 天再评估

大部分主升段，就藏在这 72 小时里

六、唯一允许卖出的“绿灯清单”

只有满足 至少 2 条，才允许卖：

□ 趋势锚明确破位

□ 行业主线切换

□ 政策 / 逻辑被证伪

□ 已先减仓 ≥50%

□ 卖出后资金立刻进入同逻辑更优标的

否则：
👉 一律视为“提前下车”

七、最后一道心理闸门（必须自问）

卖出前，必须问自己一句话（说出口）：

“如果它 3 天内涨 15%，我认吗？”

不认 → 不卖

认 → 执行卖出，不后悔
