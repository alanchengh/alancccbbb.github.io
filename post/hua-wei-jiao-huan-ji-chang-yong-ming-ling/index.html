华为交换机常用命令

2026-06-01 华为交换机

华为交换机常用命令：
dis cu
interface G0/0/44
[Huawei-GigabitEthernet1/0/1] shutdown
[Huawei-GigabitEthernet1/0/1] undo shutdown
[Quidway-GigabitEthernet0/0/4]dis this
dhcp snooping trusted
dhcp snooping enable
port-isolate enable
undo port-isolate enable
undo dhcp snooping enable
dis mac-address d297-7f72-b60a
dis mac-address g0/0/12
dis dhcp snooping confiddiguration
dis current-configuration interface g0/0/34 #查看34端口所有配置
clear configuration interface g0/0/46 #清除端口所有配置（清除后重新使用端口要undo shutdown)
undo shutdown
display ip routing-table

配置接口Eth2/0/0的端口隔离功能。

system-view
[Huawei] interface ethernet 2/0/0
[Huawei-Ethernet2/0/0] port-isolate enable group 1
注意事项
缺省情况下，端口隔离模式是二层隔离三层互通，若需要配置二三层都隔离可以执行port-isolate mode all命令配置。
同一端口隔离组的端口之间互相隔离，不同端口隔离组的端口之间不隔离。如果不指定group-id参数时，默认加入的端口隔离组为1。

[Quidway-GigabitEthernet0/0/34]undo port-security mac-address sticky
[Quidway-GigabitEthernet0/0/34]undo port-security enable
[Quidway-GigabitEthernet0/0/34]undo port-isolate enable
