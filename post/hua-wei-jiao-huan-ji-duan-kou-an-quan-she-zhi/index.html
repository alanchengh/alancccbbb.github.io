华为交换机端口安全设置

2026-06-01 华为交换机

端口安全：
interface GigabitEthernet 0/0/5 # 员工工位接口
port-security enable # 接口开启安全
port-security mac-address sticky #启用Sticky MAC功能（自动把动态MAC绑定）
port-security max-mac-num 1 # 最多允许1个MAC地址接入
port-security protect-action restrict # 超过数量，接口自动丢包+告警（防私接）
port-security mac-address sticky 7486-e20a-af51 vlan 1 # 绑定员工电脑的MAC（静态MAC）
display mac-address sticky #查看安全端口MAC
undo port-security mac-address sticky b4a9-fc88-7d11 vlan 1 #删除绑定
dis mac-address g0/0/12 #查看端口MAC
display mac-address sticky g0/0/46 #查看端口静态MAC
dis this #查看端口设置
clear configuration interface g0/0/46 #清除端口所有配置

在GE0/0/1接口下配置一条静态安全MAC地址表项。
system-view
[HUAWEI] interface gigabitethernet 0/0/1
[HUAWEI-GigabitEthernet0/0/1] port-security enable
[HUAWEI-GigabitEthernet0/0/1] port-security mac-address sticky
[HUAWEI-GigabitEthernet0/0/1] port-security mac-address sticky 8038-9691-60a3 vlan 1
