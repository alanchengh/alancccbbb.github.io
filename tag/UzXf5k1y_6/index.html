华为交换机

根据你目前提供的信息，我给你整理一份 S5700-28P-LI-AC + OpenWrt + AC + PoE交换机 + AP 的完整基础配置模板。

一、网络规划

VLAN10 Office 192.168.10.0/24
VLAN20 Finance 192.168.20.0/24
VLAN30 Printer 192.168.30.0/24
VLAN40 CCTV 192.168.40.0/24
VLAN60 Guest WiFi 192.168.60.0/24
VLAN99 Manage 192.168.99.0/24
VLAN254 Transit 192.168.254.0/30

二、S5700配置

进入系统视图：
system-view

创建VLAN
vlan batch 10 20 30 40 60 99 254

创建网关接口

ip address 192.168.10.1 255.255.255.0
dhcp select global

ip route-static 0.0.0.0 0 192.168.254.1

四、端口规划
G0/0/1-10 办公
interface range GigabitEthernet0/0/1 to GigabitEthernet0/0/10
port link-type access
port default vlan 10

G0/0/11-15 财务
interface range GigabitEthernet0/0/11 to GigabitEthernet0/0/15
port link-type access
port default vlan 20

G0/0/16-18 打印机
interface range GigabitEthernet0/0/16 to GigabitEthernet0/0/18
port link-type access
port default vlan 30

G0/0/19-22 摄像头
interface range GigabitEthernet0/0/19 to GigabitEthernet0/0/22
port link-type access
port default vlan 40

G0/0/24 上联PoE交换机
interface GigabitEthernet0/0/24
port link-type trunk
port trunk allow-pass vlan 10 60 99

G0/0/25 AC
interface GigabitEthernet0/0/25
port link-type access
port default vlan 99

G0/0/28 OpenWrt
interface GigabitEthernet0/0/28
port link-type access
port default vlan 254

五、Guest WiFi隔离

ACL：
acl number 3001
rule 5 deny ip source 192.168.60.0 0.0.0.255 destination 192.168.10.0 0.0.0.255
rule 10 deny ip source 192.168.60.0 0.0.0.255 destination 192.168.20.0 0.0.0.255
rule 15 deny ip source 192.168.60.0 0.0.0.255 destination 192.168.30.0 0.0.0.255
rule 20 deny ip source 192.168.60.0 0.0.0.255 destination 192.168.40.0 0.0.0.255
rule 25 deny ip source 192.168.60.0 0.0.0.255 destination 192.168.99.0 0.0.0.255
rule 30 deny ip source 192.168.60.0 0.0.0.255 destination 192.168.254.0 0.0.0.3
rule 100 permit ip

绑定：
interface Vlanif60
traffic-filter inbound acl 3001

六、管理员访问管理网

管理员电脑：
192.168.10.100

ACL：
acl number 3002
rule 5 permit ip source 192.168.10.100 0 destination 192.168.99.0 0.0.0.255
rule 10 deny ip source 192.168.10.0 0.0.0.255 destination 192.168.99.0 0.0.0.255
rule 100 permit ip

绑定：
interface Vlanif10
traffic-filter inbound acl 3002

八、管理IP建议

部署完成后，建议执行：
save
display current-configuration

并导出配置备份。这样后续增加 NAS、服务器、更多 AP 时，只需新增 VLAN 和 ACL，不需要重构网络。

下级还有一台二层交换机呢？

情况1：二层交换机下面只有一种业务（最简单）
例如整个二层交换机下面都是办公电脑：

二层交换机
├─ PC
├─ PC
├─ PC

那么 S5700 到二层交换机：

interface GigabitEthernet0/0/23
port link-type access
port default vlan 10

整个交换机下面：
全部 VLAN10

情况2：二层交换机下面有多种业务（推荐）

二层交换机
├─ PC VLAN10
├─ 财务PC VLAN20
├─ 打印机 VLAN30

那么：

S5700上联口
interface GigabitEthernet0/0/23
port link-type trunk
port trunk allow-pass vlan 10 20 30 40 99

二层交换机上联口
interface GigabitEthernet0/0/1
port link-type trunk
port trunk allow-pass vlan 10 20 30 40 99

二层交换机用户口

办公电脑：
interface GigabitEthernet0/0/2
port link-type access
port default vlan 10

财务：
interface GigabitEthernet0/0/3
port link-type access
port default vlan 20

打印机：
interface GigabitEthernet0/0/4
port link-type access
port default vlan 30

监控：
interface GigabitEthernet0/0/5
port link-type access
port default vlan 40

管理IP怎么配？
如果二层交换机支持管理（Web/SSH）：

例如：
管理VLAN = 99
IP = 192.168.99.3
GW = 192.168.99.1

华为一般：
vlan 99
interface Vlanif99
ip address 192.168.99.3 255.255.255.0

默认网关：
ip route-static 0.0.0.0 0 192.168.99.1
或者部分型号是：
ip default-gateway 192.168.99.1

注意 VLAN254
二层交换机上联 Trunk：
port trunk allow-pass vlan 10 20 30 40 99
不要带：
VLAN254
因为：
VLAN254
仅用于
OpenWrt
↔
S5700

存在：
S5700
PoE交换机
二层交换机
AC
AP
